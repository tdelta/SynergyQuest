How to run CoOp Dungeon
=======================

If you are on Linux, open `CoOpDungeonLinux/CoOpDungeon.x86_64`

If you are on Windows, open `CoOpDungeonWindows/Mobile-ControlledCooperativeMultiplayerGame.exe`

Troubleshooting
===============

# You can not connect with your phone browser to the address shown in the game?

-> Wait for a few seconds. A button "Not Working?" will appear. Press it and try out the additional addresses shown there

# The phone controls do not work correctly or are displayed incorrectly?

-> Make sure to use the latest version of Google Chrome on your phone.
